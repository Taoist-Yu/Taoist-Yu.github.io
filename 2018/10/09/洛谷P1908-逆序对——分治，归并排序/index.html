[洛谷P1908]逆序对——分治，归并排序

本题采用分治的思想，进而想到归并排序

先将数组划分成左右两个子区间，然后可知：

逆序对的总数 = 左区间逆序对数目+右区间逆序对数目+同时位于左右区间的逆序对的数目（_即X在左区间，Y在右区间的逆序对X,Y _）

对于左右区间的逆序对数，我们可以通过递归的方法求出，现在的问题就在于，如何求出同时位于左右区间的逆序对数目。

/类似于归并排序的合并过程/
//左区间[l,m)右区间[m,r)
//p1,p2,p分别为左右区间和辅助数组上的游标
int p1=l,p2=m,p=l;
while(p1<m && p2<r){
if(a[p1] <= a[p2]){
s[p++] = a[p1++];
} else {
ans += (m-p1);
s[p++] = a[p2++];
}
}
//以下就和普通的归并排序合并过程一样
while(p1<m)s[p++] = a[p1++];
while(p2<r)s[p++] = a[p2++];

for(p=l;p<r;p++)a[p] = s[p];
